# ts-package-skeleton
A demo of how to create a npm package using modern TypeScript

## Setup
1. clone
1. yarn install
1. yarn [command] where command is one of 
   * "test",
   * "build",
   * "lint",
   * "prepare",
   * "prepublishOnly",
   * "preversion",
   * "version",
   * "postversion"
